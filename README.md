# ✈️ Flight Reservation System API

A backend REST API for managing flight reservations, passengers, tickets, and seat assignments built with ASP.NET Core.

## 🚀 Features
- Passenger management
- Flight management
- Seat selection and assignment
- Ticket booking system
- Reservation lifecycle (create, update, cancel)
- Role-based access control (Admin/User)
- Clean layered architecture (N-Tier)

## 🛠️ Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- FluentValidation
- Repository & Unit of Work pattern
- JWT Authentication (if implemented)

## 🧱 Architecture
- Presentation Layer (API)
- Business Layer (Services)
- Data Access Layer (Repositories)
- Core Layer (Entities & DTOs)

## 📌 Main Features Flow
1. User registers/logs in
2. Searches flights
3. Selects seat
4. Creates reservation
5. Ticket is generated automatically

## ⚙️ Key Concepts Used
- Clean Architecture
- SOLID Principles
- DTO Mapping
- Exception Handling Middleware
